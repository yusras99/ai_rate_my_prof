# flashcard_saas

Project 4 of Headstarter Fellowship - Flashcards creation

### Required Modules

```bash
npm install
npm install react-markdown
npm install @mui/material @emotion/react @emotion/styled @pinecone-database/pinecone @vercel/analytics openai
npm install node
```

### Build

```bash
npm run build
```

### Development Server

```bash
npm run dev
```
